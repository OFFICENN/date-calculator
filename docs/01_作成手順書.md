# 年齢・日数計算アプリ 作成手順書

## 概要

Cursor + Vite + React + TailwindCSS で年齢・日数計算Webアプリを作成し、
GitHub + Vercel で公開するまでの手順をまとめたものです。

---

## 使用ツール

| ツール | 用途 |
|---|---|
| Cursor | コード作成・編集（AI搭載IDE） |
| Vite | 開発サーバー・ビルドツール |
| React + TypeScript | UIフレームワーク |
| TailwindCSS v4 | スタイリング |
| GitHub | ソースコード管理 |
| Vercel | ホスティング・自動デプロイ |

---

## STEP 1：プロジェクト作成

```bash
npm create vite@latest date-calculator -- --template react-ts
cd date-calculator
npm install
```

## STEP 2：TailwindCSS インストール

```bash
npm install -D tailwindcss @tailwindcss/vite
```

### vite.config.ts に追加

```ts
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";
import tailwindcss from "@tailwindcss/vite";

export default defineConfig({
  plugins: [react(), tailwindcss()],
});
```

### src/index.css の先頭に追加

```css
@import "tailwindcss";
```

## STEP 4：動作確認

```bash
npm run dev
# → http://localhost:5173/ で確認
```

## STEP 5：GitHubにリポジトリ作成 & Push

### 5-1. GitHubでリポジトリ作成

1. https://github.com/new を開く
2. Repository name: `date-calculator`
3. Public を選択
4. README等のチェックはすべて外す
5. 「Create repository」をクリック

### 5-2. ローカルからPush

```bash
git init
git add -A
git commit -m "feat: 年齢・日数計算Webアプリ初期実装"
git remote add origin https://github.com/あなたのユーザー名/date-calculator.git
git branch -M main
git push -u origin main
```

## STEP 6：Vercelでデプロイ

1. https://vercel.com/ にGitHubアカウントでログイン
2. 「Add New Project」をクリック
3. `date-calculator` リポジトリを選択して「Import」
4. 設定はデフォルトのまま「Deploy」をクリック
5. 1〜2分で公開完了
6. 公開URL（例: `https://date-calculator-xxxxx.vercel.app`）が発行される

## STEP 7：以降の更新方法

コードを修正したら以下を実行するだけで自動デプロイされる：

```bash
git add -A
git commit -m "変更内容の説明"
git push
```

Vercelが自動検知して再ビルド・再デプロイしてくれる。

## 実装済み機能一覧

- **年齢計算**: 生年月日から年齢（年・月・日）、経過日数・週数・月数を表示
- **日数計算**: 2つの日付間の差を日数・週数・時間・分で表示
- **日付加算**: 基準日に日・週・月・年を加算/減算して結果を表示
- **記念日計算**: 基準日から100日・1000日・10000日等の記念日一覧を表示
- **星座・干支**: 生年月日から自動判定
- **次の誕生日カウントダウン**: 誕生日まであと何日か表示
- **よく使う計算ショートカット**: 1週間後・30日後・1年後等をワンタップで計算
- **ダークモード**: システム設定追従 + 手動トグル
- **ユーザー名カスタマイズ**: 環境変数固定 / URLパラメータ / モーダル設定
- **レスポンシブ対応**: PC・スマホ

## 補足：主要な技術ポイント

- TailwindCSS v4 は `@tailwindcss/vite` プラグインで統合（v3以前とは設定方法が異なる）
- ダークモードは `class` ベース（HTMLのルートに `dark` クラスを付与）
- 環境変数は `VITE_` プレフィックスが必須（Viteの仕様）
- `vercel.json` でビルドコマンドと出力ディレクトリを明示指定
- 日付計算は純粋なJavaScript Date APIで実装（外部ライブラリ不要）
- タブ切替でUIを分割し、機能ごとに独立したコンポーネントとして管理
