操作説明

タブ切替

画面上部に4つのタブがあります。タップで切り替えできます。

タブ機能
🎂 年齢計算生年月日から年齢・経過日数を計算
📅 日数計算2つの日付間の差を計算
➕ 日付加算基準日に日数を加算・減算
🎉 記念日生誕1000日等の記念日一覧

年齢計算

  1. 「生年月日」欄に誕生日を入力します。
  2. 「計算」ボタンをクリックします。
表示項目内容
現在の年齢○歳○ヶ月○日
生きた日数出生日からの日数
生きた週数 / 月数出生日からの週数・月数
次の誕生日日付と曜日、あと○日
星座 / 干支自動判定して表示

日数計算

  1. 「開始日」と「終了日」を入力します（終了日のデフォルトは今日）。
  2. 「日数を計算」ボタンをクリックします。

日数・年月日・週数・時間・分の形式で表示されます。

日付加算

  1. 「基準日」を入力します（デフォルトは今日）。
  2. 「加算」または「減算」を選択します。
  3. 数値と単位（日・週・ヶ月・年）を入力します。
  4. 「計算」ボタンをクリックします。

よく使うショートカット（1週間後、30日後、半年後、1年後 等）も用意されています。

記念日計算

  1. 「基準日」に誕生日や記念日を入力します。
  2. 「計算」ボタンをクリックします。

100日目、365日目、1000日目、10000日目など、主要な記念日の日付が一覧で表示されます。各記念日には「済」（過去）または「これから」（未来）のラベルが表示されます。

ダークモード

画面右上の丸いボタンで切り替えできます。初回アクセス時はお使いの端末のシステム設定に自動追従します。手動で切り替えた場合、次回以降もその設定が記憶されます。
